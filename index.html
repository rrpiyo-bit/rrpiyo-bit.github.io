WariCan！
レトロゲーム風 割り勘計算機
▶ DESCRIPTION

飲み会・旅行・ランチなど、割り勘のシーンを 8bitレトロゲームの世界に変えた計算アプリ。 コインが弾むアニメーションとピクセルアートな UIで、面倒な計算をたのしく素早く！ 端数の切り上げ・切り捨て設定にも対応。 PWA対応でスマホにインストールして オフラインでも使えます。

▶ HOW TO PLAY
  1. まず「人数」を＋／－ボタンで設定します。 1人から最大20人まで対応。
  2. 「合計金額」欄に支払い総額を入力します。 テンキー入力・直接入力どちらもOK。
  3. 「端数処理」を「切り上げ」「切り捨て」 「四捨五入」から選択。シーンに合わせて 使い分けよう！
  4. 「CALC！」ボタンを押すと1人あたりの 金額がコインエフェクト付きで表示。 幹事は合計も確認できます。
  5. スマホのホーム画面に追加すれば アプリとして使えてさらに便利！

FOCUS QUEST
RPGポモドーロタイマー
▶ DESCRIPTION

作業をRPGのクエストに見立てたポモドーロ タイマーアプリ。25分の集中タイム＝クエスト クリア！達成するたびに★が積み上がり、 経験値（EXP）がたまっていく達成感が モチベーションを維持します。 勉強・仕事・創作活動など、集中が必要な あらゆるシーンにおすすめ。

★ HOW TO PLAY
  1. 画面上部でクエスト（作業）の内容を 入力します。何のために集中するか 明確にすることで効果アップ！
  2. フェーズタブで「WORK（25分）」 「BREAK（5分）」「LONG BREAK（15分）」 を切り替えられます。
  3. 「▶ START」ボタンでカウントダウン開始。 タイマーが0になると音で通知。 25分の集中タイムが1クエストクリア！
  4. クエストをクリアするたびに★マークが 増え、EXPが加算されます。累計の 集中時間もログで確認できます。
  5. 4回クエストをクリアしたら LONG BREAKで十分に休憩。 これを繰り返してレベルアップ！
